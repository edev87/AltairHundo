Hundo: A Coding Challenge

Live life one loop at a time! Hundo is a Javascript coding challenge using loops. The objective is to write a program which diplays the even numbers from 0 to 100, with the even numbers in BOLD.

Instructions

Enter a starting value and an ending value. Press the button to display the numbers on the page!
